# 📝 toDone (투던)
> **당신의 하루를 가볍고 완벽하게 만들어줄 상태 표시줄(트레이) 할 일 관리자**

toDone은 복잡한 창을 띄울 필요 없이, Mac의 메뉴바나 Windows의 작업 표시줄에서 클릭 한 번으로 빠르게 열고 닫을 수 있는 직관적인 데스크톱 투두(To-Do) 어플리케이션입니다.

---

## ✨ 주요 기능 (Features)
* ⚡ **빠른 접근성:** 트레이 아이콘을 클릭해 언제든 1초 만에 앱을 열고 닫을 수 있습니다.
* 📅 **다양한 뷰 제공:** 일간(Daily), 주간(Weekly), 반복(Recurring), 전체(Kanban) 탭으로 할 일을 입체적으로 관리합니다.
* 🔄 **스마트한 반복 작업:** 매일/매주 해야 하는 루틴을 설정하면 알아서 스케줄링됩니다.
* 🌐 **다국어 지원:** 한국어와 영어를 완벽하게 지원합니다.
* 🚀 **자동 업데이트:** 새로운 기능이 추가되면 앱 내에서 버튼 클릭 한 번으로 안전하게 업데이트됩니다.

---

## 🛡️ 개인정보 보호 및 보안 안내

**"설치할 때 위험한 프로그램이라고 경고가 뜨는데, 안전한가요?"**

네, 100% 안전합니다! 
toDone은 개인 개발자가 배포하는 무료 오픈소스 프로그램으로, 연간 수십만 원이 드는 공식 인증(Apple/Windows Code Signing)을 거치지 않았습니다. 이 때문에 OS에서 기본적으로 '확인되지 않은 개발자' 경고를 띄울 수 있습니다.

하지만 안심하세요. **toDone은 사용자의 어떠한 개인정보나 할 일 데이터도 외부 서버로 전송하지 않습니다.**
* **투명한 소스코드 공개:** toDone의 모든 작동 원리와 코드는 현재 보고 계신 이 GitHub 저장소에 100% 공개되어 있습니다. 전 세계 누구나 코드를 직접 읽고 검증할 수 있으므로, 악의적인 코드나 백도어가 숨겨져 있을 위험이 전혀 없습니다.
* **100% 로컬 저장:** 사용자가 작성한 모든 할 일과 설정 데이터는 오직 사용자의 PC 하드디스크 내부에만 안전하게 보관됩니다. (오프라인에서도 완벽하게 동작합니다.)
* **외부 통신 최소화:** 오직 '새로운 버전 업데이트가 있는지' 확인하기 위해 GitHub의 공식 릴리즈 서버와만 통신하며, 그 외의 어떠한 트래커나 백그라운드 데이터 수집도 존재하지 않습니다.

## 📦 다운로드 및 설치 방법 (Installation)

최신 버전 다운로드: [Releases 페이지로 이동](https://github.com/seunghok22/toDone/releases/latest)

### 🍎 macOS 사용자 (경고 우회 방법)
1. 다운로드한 `.dmg` 파일을 열어 `toDone`을 `Applications(응용 프로그램)` 폴더로 드래그합니다.
2. 앱을 처음 실행하면 **"확인되지 않은 개발자가 배포했기 때문에 열 수 없습니다"**라는 경고가 뜹니다.
3. **[해결법]:** `toDone` 앱 아이콘을 마우스 **우클릭(또는 Control + 클릭)** 한 뒤 **[열기]**를 누릅니다.
    - 시스템 설정 > 개인정보 보호 및 보안에서 '확인되지 않은 개발자'를 허용합니다. > Mac을 보호하기 위해 'toDone'을 차단했습니다. 항목에서 "그래도 열기"를 클릭합니다.

### 🪟 Windows 사용자 (경고 우회 방법)
1. 다운로드한 `.exe` 또는 `.msi` 파일을 실행합니다.
2. 파란색 화면으로 **"Windows PC 보호 (SmartScreen)"** 경고가 나타날 수 있습니다.
3. **[해결법]:** 창 중간에 있는 **[추가 정보]** 글씨를 클릭한 뒤, 우측 하단에 나타나는 **[실행]** 버튼을 누르면 안전하게 설치가 진행됩니다.
