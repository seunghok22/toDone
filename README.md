# 📝 toDone
> **당신의 하루를 가볍고 완벽하게 만들어줄 상태 표시줄(트레이) 할 일 관리자**

toDone은 복잡한 창을 띄울 필요 없이, Mac의 메뉴바나 Windows의 작업 표시줄에서 클릭 한 번으로 빠르게 열고 닫을 수 있는 직관적인 데스크톱 투두(To-Do) 어플리케이션입니다.

## ✨ 주요 기능 (Features)
* ⚡ **언제 어디서나 빠른 접근성:** 데스크톱 트레이 아이콘과 모바일 웹(PWA) 앱을 통해 1초 만에 즉각적으로 접근할 수 있습니다.
* 🎯 **시각적이고 유연한 태스크 관리:** 3단계 우선순위(High/Medium/Low) 설정과 색상 구분, 마감일 초과 작업에 대한 직관적인 경고 UI를 제공합니다. 마감일이 없는(No Deadline) 상시 작업도 깔끔하게 관리할 수 있습니다.
* 🔍 **실시간 검색 기능:** 제목과 내용을 입력하는 즉시 원하는 태스크를 빠르게 찾아냅니다.
* 🧹 **스마트한 자동화 및 캘린더:** 완료 후 일정 기간(7일/30일)이 지난 작업은 자동으로 정리되며, 공휴일과 주말이 명확히 표시된 캘린더로 일정을 한눈에 파악합니다.
* 🪶 **가벼운 리소스 점유 (데스크톱):** Tauri(Rust) 프레임워크 기반으로 제작되어 백그라운드 대기 시 100MB 미만, 앱 활성화 시 200MB 미만의 메모리만 사용합니다.
* 🔄 **반복 스케줄링:** 매일/매주 해야 하는 루틴을 설정하면 알아서 다음 일정으로 갱신됩니다.
* 🌐 **다국어 및 OS 최적화:** 한국어와 영어를 완벽하게 지원하며, macOS와 Windows 등 각 운영체제에 최적화된 맞춤형 렌더링(투명도 등)을 제공합니다.

## 🛡️ 개인정보 보호 및 보안 안내

**"설치할 때 위험한 프로그램이라고 경고가 뜨는데, 안전한가요?"**

네, 100% 안전합니다! 
toDone은 개인 개발자가 배포하는 무료 오픈소스 프로그램으로, 연간 수십만 원이 드는 공식 인증(Apple/Windows Code Signing)을 거치지 않았습니다. 이 때문에 OS에서 기본적으로 '확인되지 않은 개발자' 경고를 띄울 수 있습니다.

하지만 안심하세요. **toDone은 사용자의 어떠한 개인정보나 할 일 데이터도 무단으로 수집하지 않습니다.**
* **투명한 소스코드 공개:** toDone의 모든 작동 원리와 코드는 현재 보고 계신 이 GitHub 저장소에 100% 공개되어 있습니다. 전 세계 누구나 코드를 직접 읽고 검증할 수 있습니다.
* **로컬 기반 데이터 저장:** 사용자가 작성한 기본 데이터는 기기 내부에 안전하게 보관되며 오프라인에서도 동작합니다. (클라우드 동기화 기능 사용 시에만 암호화되어 안전하게 전송됩니다.)
* **외부 통신 최소화:** 앱 내부에서 이루어지는 통신은 버전 업데이트 확인 및 사용자가 허용한 동기화 작업에만 한정됩니다.

## 📦 플랫폼별 이용 및 설치 방법 (Installation)

### 📱 모바일 및 웹 (PWA)
별도의 설치 과정 없이 웹 브라우저 접속만으로 이용 가능하며, 홈 화면에 추가하여 네이티브 앱처럼 사용할 수 있습니다.
* **웹 접속 주소:** https://to-done-mobile.vercel.app/

### 💻 데스크톱 (최신 버전 다운로드)
* [Releases 페이지로 이동](https://github.com/seunghok22/toDone/releases/latest)

**🍎 macOS 사용자 (경고 우회 방법)**
1. 다운로드한 `.dmg` 파일을 열어 `toDone`을 `Applications(응용 프로그램)` 폴더로 드래그합니다.
2. 앱을 처음 실행하면 **"확인되지 않은 개발자가 배포했기 때문에 열 수 없습니다"**라는 경고가 뜹니다.
3. **[해결법]:** `toDone` 앱 아이콘을 마우스 **우클릭(또는 Control + 클릭)** 한 뒤 **[열기]**를 누릅니다.
    - 시스템 설정 > 개인정보 보호 및 보안에서 '확인되지 않은 개발자'를 허용합니다. > Mac을 보호하기 위해 'toDone'을 차단했습니다. 항목에서 "그래도 열기"를 클릭합니다.

**🪟 Windows 사용자 (경고 우회 방법)**
1. 다운로드한 `.exe` 또는 `.msi` 파일을 실행합니다.
2. 파란색 화면으로 **"Windows PC 보호 (SmartScreen)"** 경고가 나타날 수 있습니다.
3. **[해결법]:** 창 중간에 있는 **[추가 정보]** 글씨를 클릭한 뒤, 우측 하단에 나타나는 **[실행]** 버튼을 누르면 안전하게 설치가 진행됩니다.
