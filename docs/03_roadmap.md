## 개발 마일스톤

- **Phase 1: 초기 세팅 및 DevOps (완료)**
    - Tauri + React(Vite) + Tailwind CSS 프로젝트 생성.
    - GitHub Repository 연동 및 CI/CD(Mac/Win 빌드 및 Release) 파이프라인 구축.
- **Phase 2: UI 뼈대 및 컴포넌트 개발 (완료)**
    - 스티치(Stitch) MCP를 연결하여 UI 컨텍스트 로드.
    - Atomic 디자인 패턴에 따라 버튼, 입력창 등 기초 컴포넌트 구현.
- **Phase 3: 로컬 데이터베이스 연동 (완료)**
    - Tauri Plugin SQL (SQLite) 설치 및 테이블 스키마 생성.
    - 프론트엔드와 Rust 간의 IPC(`invoke`) 통신 로직 및 CRUD 기능 구현.
- **Phase 4: 트레이 앱(메뉴바) 동작 및 OS 최적화 (완료)**
    - 메인 윈도우 프레임 제거 및 상단 메뉴바 아이콘(Tray) 연결.
    - 창 숨김(Blur) 로직 및 백그라운드 최적화.
- **Phase 5: 일간(Daily) 및 전체(All) 칸반 보드 고도화 (완료)**
    - 3단계 상태를 가진 칸반 보드 UI 및 드래그 앤 드롭(DnD) 구현.
- **Phase 6: 전역 캘린더 스트립 및 주간(Weekly) 뷰 구현 (완료)**
    - 상단 스크롤 캘린더 스트립 바(Bar) 도입 및 각 날짜별 데이터 렌더링.
- **Phase 7: 작업 상세 모달 및 스마트 반복(Recurring) 1차 로직 (완료)**
    - 통합 모달(Modal) 팝업 구현.
    - 완료 시 다음 주기로 자동 스케줄링.
- **Phase 8: UX 폴리싱 및 완전 자동화 스케줄링 (완료)**
    - Recurring(반복 작업) 전용 탭 분리.
    - 앱 화면 포커스 및 백그라운드 타이머를 활용한 과거 방치 작업 징검다리 스케줄러 도입.
- **Phase 9: 최종 릴리즈 및 배포 (진행 중)**
    - 최종 QA 후 GitHub Actions를 통한 v1.0.0 공식 배포.
